## 一、新增模块：商办空间租赁推荐

### 模块定位
> 企业在政企通上查完政策、找到代办、选好软件，最后一步就是"在哪办公"。政企通把企业从"想开公司"到"开门营业"的整个链条全部覆盖。

### 用户流程

```
输入: 所在地(南通开发区) + 行业(能源化工)
  ↓
结果页新增"找办公室"Tab
  ↓
按需求筛选:
  ├─ 类型: 独立办公室 / 开放工位 / 共享办公 / 整层出租 / 园区入驻
  ├─ 面积: 10-50㎡ / 50-200㎡ / 200-500㎡ / 500㎡以上
  ├─ 价格: 1000以下 / 1000-3000 / 3000-5000 / 5000以上
  └─ 配套: 注册地址 / 会议室 / 停车 / 食堂 / 交通
  ↓
商办空间详情页: 实拍照片、平面图、价格、配套、周边、评价
  ↓
"预约看房" / "在线咨询" → 撮合成交 → 平台抽佣
```

## 二、商办空间分类体系

### 空间类型

| 类型 | 适合企业 | 典型诉求 |
|---|---|---|
| 独立办公室 | 3-20人团队，需要私密空间 | 隔音、独立空调、企业形象 |
| 开放工位 | 1-5人创业团队、自由职业者 | 低成本、灵活、拎包入驻 |
| 共享办公/联合办公 | 初创公司、外地办事处 | 社区氛围、配套齐全、短租 |
| 小型套间 | 5-15人，需独立门面 | 独立入口、招牌展示 |
| 整层/整栋 | 成熟企业、需要独栋办公楼 | 企业总部形象、园区政策 |
| 虚拟地址 | 仅需注册地址，不实际办公 | 工商注册刚需 |

### 信息来源

| 来源 | 类型 | 独家性 |
|---|---|---|
| **你企服公司1000平自持** | **独立办公室+开放工位** | **自有资源，零成本** |
| 南通开发区科技园区 | 政府招商载体 | 企服合作关系，一手信息 |
| 开发区各写字楼业主 | 商业楼宇 | 招商入驻，收推广费 |
| 南通全市商办中介平台 | 全市房源 | 合作/API对接 |
| 个人房东直租 | 小面积独立办公室 | 平台入驻 |

### 空间信息字段

| 字段 | 说明 |
|---|---|
| 名称/地址 | 楼宇名称+详细地址 |
| 空间类型 | 独立办公室/工位/共享/整层 |
| 面积 | 精确到㎡ |
| 月租金 | 元/月，注明是否含物业/水电/网络 |
| 押金方式 | 押一付三/押一付一 |
| 最短租期 | 1个月/3个月/半年/一年 |
| 可注册 | 是否支持工商注册地址 |
| 配套 | WiFi、空调、会议室、停车、食堂、地铁距离 |
| 实拍照片 | 至少3张（全景+工位+公共区） |
| 平面图 | 可选 |
| 周边 | 银行、餐饮、政务大厅距离 |
| 园区政策 | 该楼宇/园区是否有租金补贴（这是你的独家信息优势） |

## 三、信息整合——园区政策自动关联（核心壁垒）

这是政企通做商办租赁最狠的一手：**不只看房源，还能自动匹配该房源所在园区的租金补贴政策。**

```
用户看中南通开发区某写字楼一间办公室，月租5000元
  ↓
政企通自动弹出:
  "该房源位于南通开发区科技企业加速器，符合条件的企业可享受:
   ✅ 科技型中小企业: 租金补贴50%，最高20万/年
   ✅ 首次入驻开发区: 免租3-6个月
   ✅ 招商引荐奖励: 落地2000万以上企业可获引荐奖励
  → 您的企业预计实际月租仅需2500元"
```

这个功能别人做不了——园区内部补贴政策只有你跟开发区管委会对接的一手渠道能拿到。

## 七、招商指标联动（一鱼多吃）

这个模块还能帮你**完成企服公司今年的招商任务**：

```
企业通过政企通找到办公场地
  → 实地落户南通开发区
  → 工商注册委托企服公司代办
  → 该企业达到2000万入库标准
  → 计入企服公司招商KPI
  → 同时拿政府招商奖励(5-30万/家)
```

一套动作，赚了租金差价、代办费、招商奖励、平台撮合费，四重收益。
