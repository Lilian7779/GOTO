# 政企通 —— 政企信息服务与精准对接平台 产品规划书 V2.0

**版本**：MVP v2.0
**定位**：输入"所在地 + 行业"，精准返回政府侧所有相关——政策/办事/招投标/监管/人脉/补贴
**核心逻辑**：两维交叉定位（区域×行业）→ 政企信息精准分发 → 付费深度服务转化
**持有公司**：_______（南通）信息服务有限公司

---

## 一、为什么是"政企通"而非"惠企通"

| | 惠企通 | 政企通 |
|---|---|---|
| 范围 | 仅惠企政策 | 政策 + 办事指南 + 招投标 + 监管动态 + 政务对接 |
| 用户心智 | "来领补贴的" | "来跟政府打交道的" |
| 使用频率 | 低频（有新政策才看） | 中高频（办事、查标讯、看监管） |
| 变现深度 | 代办申报 | 代办+咨询+培训+数据服务+政企对接撮合 |
| 竞争壁垒 | 政策匹配，谁都能做 | 区级政务信息一手触达，你独有 |

## 七、变现漏斗

```
第1层: 精准信息(免费)
  → 政企通成为企业"跟政府打交道的首页"
  → 日活企业用户沉淀画像数据

第2层: 深度报告(99-299元/次)
  → 企业专属政策匹配报告 + AI解读 + 申报优先级排序
  → 转化率约5-8%

第3层: SaaS工具订阅(500-2000元/月)
  → 税务风险扫描 / 招投标监控 / 政策变动提醒
  → 转化率约2-3%

第4层: 代办服务(2000-50000元/单)
  → 高新认定 / 专精特新申报 / 危化证 / 补贴申领
  → 转化率约1-2% 但客单价高

第5层: 政企对接撮合(按效果付费)
  → 招商引荐 / 政企座谈会 / 项目申报联合体
  → 后端变现
```
