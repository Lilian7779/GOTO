# 税务风险扫描工具MVP功能清单

## 产品定义
- **产品名称**：税安通（暂定）
- **定位**：面向中小企业的月度税务风险自动扫描与预警SaaS工具
- **目标客户**：财税公司代账客户（首批）、自记账的中小企业（后续）
- **定价**：500-1000元/月/户（MVP期手工出报告500元）
- **开发策略**：手工验证 → 低代码半自动 → Web产品化

---

## MVP v1.0 功能清单（手工跑阶段，第2个月）

### 输入：客户提供哪些数据
| 数据类型 | 获取方式 | 必要性 |
|---|---|---|
| 增值税申报表（近3个月） | 财税公司已有/客户上传 | 必填 |
| 企业所得税申报表（最近季度） | 财税公司已有/客户上传 | 必填 |
| 进项发票清单 | 财税公司导出 | 建议 |
| 销项发票清单 | 财税公司导出 | 建议 |
| 财务报表（资产负债表+利润表） | 财税公司已有 | 建议 |
| 社保申报人数 | 财税公司已有 | 选填 |

### 输出：报告包含哪些模块
- [ ] **增值税风险扫描**
  - 月度税负率计算 + 行业对比
  - 进销项税额匹配度分析
  - 留抵税额合理性检查
  - 发票作废/红冲异常标记

- [ ] **企业所得税风险扫描**
  - 季度贡献率与行业对比
  - 成本费用率异常检测
  - 营业外收支占比异常
  - 预缴与汇算差额预估

- [ ] **发票合规检查**
  - 进项发票品名与经营范围匹配
  - 大额发票（>5万）逐笔抽查
  - 异地发票占比异常
  - 夜间/节假日开票异常

- [ ] **税种间勾稽验证**
  - 增值税收入 vs 企业所得税收入
  - 个税申报人数 vs 社保人数
  - 印花税计税依据 vs 购销合同额
  - 房产税/土地使用税是否漏报

- [ ] **优惠政策匹配**
  - 小微企业资格检查
  - 六税两费减半适用
  - 加计抵减/加计扣除适用
  - 地方园区优惠政策匹配

- [ ] **风险评级与整改建议**
  - 综合风险评分（百分制）
  - 紧急/关注/建议三级分类
  - 每项风险附带整改指引和政策依据

## MVP v2.0 功能清单（半自动化阶段，第3个月）

基于简道云/明道云搭建：

- [ ] **客户自助上传入口**：表单上传增值税申报表+发票清单

- [ ] **报告自动生成**：基于模板自动填充，支持PDF导出
- [ ] **客户管理后台**：客户列表、报告记录、订阅状态

## 第一批种子客户（10户，从财税公司存量筛选）

筛选标准：
1. 年营收500万以上
2. 最近12个月有过税务异常/更正申报
3. 行业非高风险（避开废钢/再生资源/灵活用工等敏感行业）
4. 企业主对税务合规有意识

客户画像示例：

| 客户类型 | 典型痛点 | 产品价值点 |
|---|---|---|
| 贸易公司（年营收800万） | 进销项商品名不一致、异地开票多 | 发票合规检查+进销匹配 |
| 制造企业（年营收2000万） | 研发费用未加计扣除、成本率偏高 | 优惠匹配+成本异常检测 |
| 建筑劳务（年营收1500万） | 个税社保人数差、农民工工资专户 | 税种勾稽+社保比对 |
| 电商企业（年营收500万） | 平台收入与申报不一致 | 收入比对+刷单风险提示 |

## 竞争壁垒

| 壁垒类型 | 具体内容 |
|---|---|
| 数据壁垒 | 通过财税公司获取真实客户数据，持续优化规则准确性 |
| 政策壁垒 | 你拥有一手税务指导信息，规则库保持"领先同行一个通知周期" |
| 渠道壁垒 | 企服+财税两家公司持续导流，获客成本近乎零 |
| 信任壁垒 | 存量客户已建立代账关系，推税安通是"升级"而非"新购" |
