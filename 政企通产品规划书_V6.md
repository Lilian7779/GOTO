# 政企通 —— 政企信息服务与精准对接平台 产品规划书 V6.0

**版本**：MVP v6.0（终版）
**定位**：输入"所在地 + 行业"，精准返回六大模块——政策匹配、财税服务商、企业软件、办公空间、**新公司注册必做清单**、招投标/办事指南
**核心逻辑**：两维交叉定位（区域×行业）→ 全链路匹配 → 付费转化
**持有公司**：_______（南通）信息服务有限公司

---

## 一、六大模块总览

| 模块 | 用户问题 | 政企通回答 | 变现 |
|---|---|---|---|
| 惠企政策 | 我能拿什么补贴？ | 按区域+行业精准匹配，含金额+截止时间+申报条件 | 深度报告199元 + 代办申报 |
| 财税服务商 | 找谁帮我办？ | 本地财税公司列表+评价，你自有公司置顶 | 入驻年费 + 撮合佣金 |
| 企业软件 | 用什么工具管公司？ | 按行业推荐软件，你自研产品（税安通等）置顶 | CPA/CPS + 厂商年费 |
| 办公空间 | 在哪办公最划算？ | 自有1000平+园区房源+自动匹配租金补贴 | 租金 + 撮合佣金 + 招商奖励 |
| 新公司指引 | 执照拿到了下一步？ | 按时间线列出必做事项，逾期预警 | 单项代办 + 年度管家套餐 |
| 招投标/办事 | 有哪些招标？去哪办事？ | 该区该行业招投标公告+办事指南 | 标书代写 + 咨询 |

## 二、新注册公司必做事项时间线

### 通用版

| 时间 | 事项 | 重要性 | 代办 |
|---|---|---|---|
| 注册前 | 核名+经营范围+注册地址+股权 | 基础 | 企服代办 |
| 第1天 | 领执照·刻章·法人实名 | ⚠️ | 企服代办 |
| 1-3天 | 税务报到（逾期罚款） | 🔴 | 500-1000 |
| 3-7天 | 银行开户 | ⚠️ | 免费预约 |
| 7-15天 | 社保/公积金开户 | ⚠️ | 300-500 |
| 30天内 | 首次纳税申报+财务制度备案 | 🔴 | 500-1000 |
| 60天内 | 行业许可证·商标·科小入库 | 🟡 | 按项目 |
| 次年1-6月 | 工商年报+汇算清缴 | 🔴 | 200-500/1000-5000 |

### 行业增强（示例：危化品经营）

额外增加：危化证办理（20-45工作日）、安全生产责任制、应急预案备案、安全管理人员持证、员工三级安全教育
