# 惠企通 —— 惠企政策智能推荐平台 产品规划书

**版本**：MVP v1.0
**定位**：面向中小企业的"政策头条"——输入企业画像，秒级匹配可申报的惠企政策
**商业模式**：免费查政策 → 付费深度服务（申报代办/合规评估/财税优化）
**技术公司**：_______（南通）信息服务有限公司

---

## 一、产品定义

### 核心功能一句话描述
> 企业填一份表单（5个字段），系统自动从政策库中筛选出"你能享受哪些政策、能拿多少钱、怎么申报、截止哪天"。

### 差异化优势
| 竞品 | 痛点 | 惠企通做法 |
|---|---|---|
| 政府部门官网 | 政策堆砌，看不懂，不知道哪个适用 | AI解读+精准匹配，只说"你能拿的" |
| 企查查/天眼查 | 查企业居多，政策匹配弱 | 专注政策推荐，行业+区域双重过滤 |
| 传统代办中介 | 人工服务贵，信息差赚差价 | 先免费匹配建立信任，再付费深度服务 |
| 各类政策匹配小程序 | 泛匹配，准确率低 | 区域深扎南通开发区起步，政策颗粒度到区级 |

---

## 二、用户流程

```
访问惠企通 → 填写企业画像(5字段) → 点击"一键匹配"
→ 返回匹配政策列表(按"能拿多少钱"排序)
→ 点击任一政策查看详情:条件解读/申报流程/截止时间/可获金额
→ 需要代办? → 跳转企服公司付费服务 / 下载申报材料模板
→ 需要财税合规? → 跳转财税公司代账/税筹 / 购买税务风险扫描工具
```

### 企业画像输入字段（MVP仅5个，门槛极低）

| 序号 | 字段 | 类型 | 说明 |
|---|---|---|---|
| 1 | 企业名称 | 文本 | 用于自动补全工商信息 |
| 2 | 所在区域 | 下拉 | 南通市 → 开发区 / 崇川区 / 通州区... |
| 3 | 所属行业 | 下拉 | 制造业/信息技术/贸易/建筑/物流... |
| 4 | 企业规模 | 单选 | 微型(<100万)/小型(100-2000万)/中型(2000-4亿) |
| 5 | 成立年限 | 单选 | <1年/1-3年/3-5年/5年以上 |

### 高级画像（V2.0扩展，默认不填）

| 字段 | 用途 |
|---|---|
| 是否有研发投入 | 匹配科技类补贴 |
| 是否高新技术企业 | 匹配高新专属政策 |
| 用工人数 | 匹配稳岗/社保补贴 |
| 近一年纳税额 | 匹配税收返还/奖励 |
| 是否有进出口 | 匹配外贸扶持 |

## 四、匹配引擎设计

### 匹配逻辑

```
政策库 × 标签体系 → 企业画像输入 → 规则引擎 → 匹配结果

规则引擎三层过滤:
┌─ 硬条件过滤(必须满足): 区域、行业、成立年限
├─ 加分条件(满足越多得分越高): 规模、是否有研发、用工等
└─ 排除条件(一票否决): 如"已享受同类补贴不可重复申报"
```

### 匹配结果展示

| 展示字段 | 说明 |
|---|---|
| 政策名称 | 链接到原文 |
| 匹配度 | ⭐⭐⭐⭐⭐ / 匹配度XX% |
| 可获金额 | "最高50万元" / "减免15%企业所得税" |
| 申报条件摘要 | 一句话提炼核心门槛 |
| 申报时间 | "常年申报" / "截至2026年X月X日" |
| 申报入口 | 跳转官方链接/企服公司代办按钮 |
| 深度服务 | "需要代办？点击咨询" → 企服公司转化入口 |

## 七、盈利模型

### 三级漏斗

```
第1层: 免费匹配（引流）
  → 日均100-500UV，沉淀企业画像数据
  ↓
第2层: 付费深度报告（99-299元/次）
  → 政策匹配报告(含AI解读) + 申报材料清单 + 时间规划表
  ↓
第3层: 企服+财税公司代办（3000-20000元/单）
  → 高新认定代办 / 专精特新申报 / 补贴申领代办 / 税筹合规
```
