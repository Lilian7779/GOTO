## 一、新增模块：企业软件推荐与评价

### 模块定位
> 企业查完政策、找到代办，第三步就是"用什么工具来管公司"。政企通不仅要告诉企业"能拿什么政策"，还要告诉企业"用什么软件把你的公司管好"。

### 用户流程

```
输入: 所在地(南通开发区) + 行业(能源化工)
  ↓
结果页新增"推荐软件"Tab
  ↓
按企业生命周期阶段推荐软件:
  ├─ 初创期: 工商注册辅助、记账软件
  ├─ 运营期: ERP/进销存/CRM/OA/财税软件
  ├─ 发展期: BI分析/项目管理/人事系统
  └─ 合规期: 安全台账/环保管理/合同审查
  ↓
软件详情页: 功能介绍、价格方案、用户评价、行业案例
  ↓
点击"免费试用"/"联系厂商" → 引导注册 → 平台收CPA佣金
```

## 三、软件评价体系

| 评价维度 | 权重 | 说明 |
|---|---|---|
| 功能匹配度 | 30% | 针对该行业的需求覆盖程度 |
| 易用性 | 20% | 上手难度、操作体验 |
| 性价比 | 20% | 功能与价格是否匹配 |
| 售后服务 | 15% | 响应速度、问题解决率 |
| 稳定性 | 15% | 崩溃/卡顿/数据丢失情况 |

### 评价来源

| 来源 | 权重 | 说明 |
|---|---|---|
| 平台用户评价 | 60% | 政企通企业用户使用后评价 |
| 行业评测 | 25% | 平台自评+第三方评测引用 |
| 工商/备案数据 | 10% | 软件厂商资质、经营状态 |
| 用户规模 | 5% | 该行业使用企业数量 |

### 防刷机制
- 仅真实用户可评价（需验证企业身份）
- 同一企业同一软件只能评价一次
- AI异常检测：同一IP、短时间批量评价触发人工审核

## 六、软件推荐变现模式

| 模式 | 收费方式 | 预估收入 |
|---|---|---|
| CPA（按试用） | 用户点击"免费试用"并注册 → 厂商付30-100元/个 | 月100次×50元=5000元 |
| CPS（按成交） | 用户购买付费版 → 平台抽佣15-30% | 月10单×2000×20%=4000元 |
| 入驻展示费 | 软件厂商在政企通建详情页 → 3000-8000元/年 | 20家×5000=10万/年 |
| 置顶推广 | 指定行业搜索结果页置顶 → 2000-5000元/月 | 5家×3000=1.5万/月 |
| 软文测评 | 平台出软件测评文章 → 1000-3000元/篇 | 4篇×2000=8000元/月 |
| **自有软件导流** | **政企通用户 → 试用税安通 → 转化付费** | **零成本获客，价值无限** |
