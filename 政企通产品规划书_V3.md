## 一、新增模块：区域财税服务商信息与评价

### 模块定位
> 政企通不只是"查政策"，更是"找对人"——企业查到政策后，下一步就是找谁代办。财税服务商评价模块解决"找哪家靠谱"的问题。

### 用户流程

```
输入: 所在地(南通开发区) + 行业(能源化工)
  ↓
政企通返回结果页，新增"本地财税服务商"Tab
  ↓
列表展示: 财税公司名称 | 评分 | 服务标签 | 距离 | "了解详情"
  ↓
点击进入详情页:
  - 公司基本信息
  - 服务范围与擅长行业
  - 用户评价(星级+文字)
  - 服务案例
  - 咨询入口(在线/电话)
  ↓
咨询 → 撮合成功 → 平台抽佣 / 年费入驻
```

### 财税公司信息字段

| 字段 | 说明 | 来源 |
|---|---|---|
| 公司名称 | 工商注册全称 | 入驻时提交 |
| 办公地址 | 区域精确到街道 | 入驻时提交 |
| 服务区域 | 覆盖哪些区县 | 入驻时勾选 |
| 擅长行业 | 能源/制造/商贸/建筑/IT等 | 入驻时勾选 |
| 服务标签 | 代理记账、税筹规划、高新申报、出口退税、土地增值税清算... | 入驻时勾选+平台审核 |
| 资质证书 | 代理记账许可证/税务师事务所证书等 | 上传审核 |
| 团队规模 | 注册会计师X人、税务师X人 | 入驻时填写 |
| 经营年限 | 成立X年 | 自动从工商数据拉取 |
| 服务企业数 | 累计服务XX家 | 入驻时填写+平台验证 |
| 评分 | 1-5星 | 用户评价综合计算 |
| 评价数 | XX条评价 | 实时累计 |
| 联系方式 | 电话/微信(脱敏展示) | 入驻时提交 |

### 评价体系设计

| 评价维度 | 权重 | 说明 |
|---|---|---|
| 专业能力 | 30% | 对政策的理解准确度、税务筹划方案质量 |
| 响应速度 | 20% | 咨询回复是否及时、申报是否按时提交 |
| 服务态度 | 15% | 沟通体验、主动提醒 |
| 成功率 | 25% | 申报/退税/认定等事项的实际通过率 |
| 性价比 | 10% | 收费合理度 |

### 评价规则（防刷）

| 规则 | 说明 |
|---|---|
| 仅真实客户可评 | 需上传服务合同或发票截图验证 |
| 一单一评 | 同一服务事项只能评价一次 |
| 平台审核 | 所有评价先审后发，过滤恶意差评/刷好评 |
| 匿名展示 | 评价者头像+姓氏脱敏展示（如"张**"） |
| 追评功能 | 服务完成后30天内可追评 |

### 变现模式

| 模式 | 收费 | 说明 |
|---|---|---|
| 基础入驻 | 免费 | 财税公司提交信息，平台审核后上线，不承诺曝光 |
| 认证入驻 | 3000-5000元/年 | 资质认证标识 + 优先展示 + 客户咨询推送 |
| 置顶推广 | 1000-3000元/月 | 指定区域+行业搜索结果置顶 |
| 商机撮合 | 成交佣金10-15% | 平台匹配客户→成交后抽佣 |
| **你自有财税公司** | **免费+固定置顶** | 南通开发区+全行业首位，自有流量不外溢 |

## 四、财税服务商模块增长飞轮

```
自有财税公司置顶(零成本获客)
  ↓
吸引其他财税公司关注政企通
  ↓
财税公司入驻 → 丰富平台供给 → 企业用户选择增多
  ↓
企业用户量增长 → 更多财税公司愿意付费入驻
  ↓
平台抽佣+入驻费成为稳定收入
  ↓
数据积累 → 哪家财税公司擅长什么行业一清二楚 → 匹配更精准
```
